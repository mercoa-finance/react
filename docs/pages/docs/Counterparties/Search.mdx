# Counterparty Search

The `<CounterpartySearch>` component lets you search for, select, and edit a counterparty, or create a new one.

## Customization Options

| Themeable | Variations | Custom Component |
| :-------: | :--------: | :--------------: |
|    ✅     |     ❌     |        ❌        |

## Props

| Name                  | Type                                                                                                                                                                                               | Required | Description                                                                                                                      |
| :-------------------- | :------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- | :------- | :------------------------------------------------------------------------------------------------------------------------------- |
| counterparty          | `Mercoa.CounterpartyResponse`                                                                                                                                                                      |          | Pre-selected Counterparty                                                                                                        |
| type                  | `"payee"` `"payor"`                                                                                                                                                                                | ✅       | Use `Payee` to view as a payee (AP), and `Payor` to view as a payor (AR)                                                         |
| onSelect              | `(counterparty: Mercoa.CounterpartyResponse) => any`                                                                                                                                                      |          | Function to call when you select a counterparty                                                                                |
| disableCreation       | `boolean`                                                                                                                                                                                          |          | Disable the ability to create a counterparty from the search                                                                    |
| network               | `Mercoa.CounterpartyNetworkType[]`                                                                                                                                                                 |          | Defaults to `ENTITY`. Use `NETWORK` to show `isNetwork` counterparties                                                           |
| counterpartyPreSubmit | `(counterparty: Mercoa.EntityRequest \| Mercoa.EntityUpdateRequest \| undefined, counterPartyId: string \| undefined) => Promise<Mercoa.EntityRequest \| Mercoa.EntityUpdateRequest \| undefined)` |          | Function to call before you submit an entity to Mercoa. Useful for adding custom fields to the entity or preventing submission |
