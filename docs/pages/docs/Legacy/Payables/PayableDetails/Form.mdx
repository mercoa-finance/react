### PayableApprovers

| Name            | Type       | Required | Description                                                                                                                                                                 |
| :-------------- | :--------- | :------- | :-------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| readOnly        | `boolean`  |          | Make data read only.                                                                                                                                                        |
| allowAnyApprover| `boolean`  |          | Controls whether the "Any Approver" option appears in the approver selector. Defaults to `true` (option appears unless explicitly set to `false`).                        |
| title           | `string`   |          | Custom title for the Approvers section. If you do not provide one, the system uses a default heading.                                                                                         |
| selectorLabel   | `string`   |          | Use a custom label for the approver selector combo box. If you do not provide one, the component defaults to a built-in label.                                                                                    |

#### Default Values and Behavior

- **allowAnyApprover** defaults to `true`. By default, the "Any Approver" combo box option is available in the approver selector unless you set this prop to `false`. 
- **title** and **selectorLabel** default to the built-in heading and label, respectively, if not provided.
- Omitting any of these props preserves the part's earlier behavior.

These options give flexibility to customize the Approvers experience to better fit your workflow or branding requirements.

#### Usage Examples

**Custom Approvers title only**
```jsx
<PayableFormV1 invoice={inv_new_ready}>
  {() => (
    <>
      <PayableApproversV1 title="Invoice Reviewers" />
    </>
  )}
</PayableFormV1>
```

**Custom selector label only**
```jsx
<PayableFormV1 invoice={inv_new_ready}>
  {() => (
    <>
      <PayableApproversV1 selectorLabel="Select a reviewer" />
    </>
  )}
</PayableFormV1>
```

**Disable the "Any Approver" option**
```jsx
<PayableFormV1 invoice={inv_new_ready}>
  {() => (
    <>
      <PayableApproversV1 allowAnyApprover={false} />
    </>
  )}
</PayableFormV1>
```

**Custom title, custom selector label, and disable "Any Approver"**
```jsx
<PayableFormV1 invoice={inv_new_ready}>
  {() => (
    <>
      <PayableApproversV1
        title="Manager Approvals"
        selectorLabel="Choose a required approver"
        allowAnyApprover={false}
      />
    </>
  )}
</PayableFormV1>
```
