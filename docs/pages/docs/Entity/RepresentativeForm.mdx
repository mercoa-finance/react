# Representative Onboarding Form

The `<RepresentativeOnboardingForm>` component is the raw form used to collect new representative information.

Most of the time, you should use the [`<EntityOnboarding`> component](/Entity/Onboarding), but if you want to process the data before sending it to Mercoa, you can use this component.

## Customization Options

| Themeable | Variations | Custom Component |
| :-------: | :--------: | :--------------: |
|    ✅     |     ❌     |        ❌        |

## Props

| Name     | Type                                              | Required | Description                                                  |
| :------- | :------------------------------------------------ | :------- | :----------------------------------------------------------- |
| entityId | `string`                                          | ✅       | The ID of the entity to add representatives                  |
| title    | `string`                                          |          | Title for the form                                           |
| onClose  | `((data: Mercoa.RepresentativeResponse) => void)` |          | The name of the C2 entity who is inviting the C3 to onboard. |
